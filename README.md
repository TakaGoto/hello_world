hello README
============

to create a vm, you need virtual box and vagrant
https://www.vagrantup.com/
https://www.virtualbox.org/

`git clone https://github.com/TakaGoto/hello_world.git`
`cd hello_world`
`bundle install`

to run chefspec:
`rspec`

to run serverspec:
`kitchen verify`

to converge:
`kitchen converge`
